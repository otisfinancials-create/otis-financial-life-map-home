I am building a personal finance web app called "Otis" (working title). It is a "Financial Life Map" — a single-pane-of-glass dashboard that combines cash flow forecasting, net worth tracking, bill management, investment monitoring, loan amortization, and life event planning into one beautifully simple product.

Target user: Households earning $160K+, currently managing their finances in a spreadsheet. They are sophisticated but want simplicity, speed, and automation. They do NOT want a bloated app — they want precision and clarity.

Tech stack:


Framework: Next.js (App Router) + Tailwind CSS
Database: PostgreSQL (managed by Replit)
AI logic: Claude API (claude-sonnet-4-6)
Financial data: Plaid API (to be integrated later)
Auth: Clerk (to be integrated later)


Design direction:


Clean, modern, and data-dense but never cluttered
Think "Bloomberg meets Notion" — professional but approachable
Dark mode preferred as the default
Mobile-first responsive layouts
The AI assistant persona is named "Otis" — friendly, smart, head-tilts while thinking
