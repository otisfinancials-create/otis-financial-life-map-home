### Backend — Server Side
Install the Plaid Node.js SDK:
npm install plaid
Create a Plaid client configuration:
javascriptconst { Configuration, PlaidApi, PlaidEnvironments } = require('plaid');

const configuration = new Configuration({
  basePath: PlaidEnvironments[process.env.PLAID_ENV],
  baseOptions: {
    headers: {
      'PLAID-CLIENT-ID': process.env.PLAID_CLIENT_ID,
      'PLAID-SECRET': process.env.PLAID_SECRET,
    },
  },
});

const plaidClient = new PlaidApi(configuration);
Create these API routes:
Route 1 — Create Link Token:
POST /api/plaid/create-link-token
This generates a temporary token that initializes Plaid Link for the user. Request body: { user_id }. Call plaidClient.linkTokenCreate() with:

user: { client_user_id: user_id.toString() }
client_name: 'Otis Financial'
products: ['transactions', 'liabilities']
country_codes: ['US']
language: 'en'
Return: { link_token }

Route 2 — Exchange Public Token:
POST /api/plaid/exchange-token
After user completes Plaid Link, exchange the public_token for a permanent access_token. Request body: { public_token, user_id }. Call plaidClient.itemPublicTokenExchange(). Store the access_token and item_id securely in a new database table:
sqlCREATE TABLE plaid_items (
  id SERIAL PRIMARY KEY,
  user_id INTEGER NOT NULL REFERENCES users(id),
  access_token TEXT NOT NULL,
  item_id TEXT NOT NULL,
  institution_id TEXT,
  institution_name TEXT,
  created_at TIMESTAMP DEFAULT NOW(),
  updated_at TIMESTAMP DEFAULT NOW()
);
Return: { success: true, item_id }
Route 3 — Fetch Accounts:
POST /api/plaid/accounts
After token exchange, immediately fetch the user's accounts. Call plaidClient.accountsGet() using the stored access_token. For each account returned, upsert into the connected_accounts table:

Map Plaid account types to Otis account types:

depository + checking → Checking
depository + savings → Savings
credit → Credit Card
investment → Investment
loan + mortgage → Mortgage
loan (other) → Loan


Store: account_id (Plaid's), name, official_name, type, subtype, current_balance, available_balance, institution_name
Mark each account as plaid_connected: true
Store the plaid_item_id reference

Return: { accounts: [...] }
Route 4 — Get Institution Name:
POST /api/plaid/institution
Given an institution_id, return the institution name and logo. Call plaidClient.institutionsGetById(). Used to show the bank name after connection.
